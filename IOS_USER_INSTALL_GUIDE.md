## Troubleshooting

### "Add to Home Screen" option not showing?
- Make sure you're using **Safari** (not Chrome)
- Check that the website has fully loaded
- Try refreshing the page

### App not opening in full screen?
- Make sure you tapped the icon on your home screen
- If it opens in Safari, try deleting and reinstalling

### App not working offline?
- Open the app at least once while connected to internet
- The app will cache necessary files for offline use
